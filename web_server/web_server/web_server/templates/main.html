Our Project

Covid-19 (Coronavirus) pandemic has been the top challenge of public health over the world since February 2020. With the first and the most number of historical confirmed cases in Australia, Victoria has suffered from several outbreaks. This platform provides insight into public sentiment and hot topics with Twitter during Covid-19 pandemic in Victoria, with comprehensive analysis encompassing the changes of economy.
